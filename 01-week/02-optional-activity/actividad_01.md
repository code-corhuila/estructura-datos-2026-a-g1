# 2. Actividad

## Aplicación elegida: Facebook

### 1. Tipos de datos que maneja:
- **Usuarios** (nombre, correo, foto de perfil, fecha de nacimiento)
- **Publicaciones** (texto, imágenes, videos, fecha de publicación)
- **Comentarios** (contenido, autor, fecha)
- **Reacciones** (me gusta, me encanta, me divierte, etc.)
- **Amigos**

### 2. Organización de los datos:

- **Usuarios:**  
  Podrían organizarse en una estructura tipo *objeto* o *registro*, donde cada usuario tenga atributos personales y listas asociadas.

- **Publicaciones:**  
  Podrían almacenarse en una *lista* dentro del perfil de cada usuario o en una lista general para el inicio (feed).

- **Comentarios:**  
  Podrían organizarse en una *lista* dentro de cada publicación.

- **Reacciones:**  
  Podrían manejarse mediante un *contador* o una estructura tipo *diccionario* que almacene el tipo de reacción y la cantidad.

- **Amigos:**  
  Podrían almacenarse en una *lista* o *conjunto* para evitar duplicados.

---

# 3. Conclusión

Facebook utiliza distintas estructuras de datos como listas, objetos, conjuntos y diccionarios para organizar la información de manera eficiente y permitir interacciones rápidas entre los usuarios.
